# Ouster SDK Packaging

This repository builds and publishes precompiled Ouster SDK binaries
for Linux, macOS, and Windows.

## Versioning

- Release versions exactly match upstream `ouster-sdk` tags.
- Example: `v0.10.0`

## Artifacts

Each release contains:
- headers
- static libraries
- CMake config files

## Supported Platforms

- Linux x86_64
- macOS arm64
- Windows MSVC x64
